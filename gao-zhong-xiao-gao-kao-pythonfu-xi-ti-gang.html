一、Python的运行和控制

程序运行逻辑

在Python中没有函数的层级关系，例如主函数和其他函数等关系（主函数与用户定义函数之间的关系）。

示例程序：

def output():
    print("Hello")
    return

output()

执行结果：

Hello

程序分析：

def为用户定义函数。Python模糊主函数概念，用户定义函数在定义之后才可引用。

输出Hello World——Print函数

示例程序

print("Hello")

执行结果：

Hello

程序分析

print，打印函数，用于输出字符，数据变量等信息 弱化int，char等变量

Print函数语法格式

输出规定字符，文字等内容

print("Hello")
print("12345")
print("你好")

输出变量值等内容

x = 10
print(x)

混合一下（输出输入的值——input函数）

x = input("Please input a number")
print("Your input number is",x)

输入字符，数字等——input函数

示例程序

x = input("Please input")
print("Your input is",x)

程序分析

  1. 程序运行逻辑是本身为主函数（Python中不强调主函数概念——既可以面向对象又可以面向过程）
  2. 无需分析输入的数值类型（Python中弱化int char等概念）

判断语句——if……else……

示例程序

标准语句

x = int（input("Please input a number")）
if x == 1:
    print("You are right")
else:
    print("Maybe there is something Wrong")

执行结果

输入1：输出 You are right

输入非1的内容：输出 Maybe there is something wrong

添加elif

x = int（input("Please input a number")）
if x == 1:
    print("You are right")
elif x == 2:
    print("Maybe you are right")
else:
    print("Maybe there is something Wrong")

执行结果

输入1：输出 You are right

输入非1的内容：输出 Maybe there is something wrong

输入2：输出 Maybe you are right

程序解读

程序的运行方式

  • 满足if的条件，执行下方内容
  • 不满足if,但是满足elif,执行elif，执行elif下方语句
  • 不满足，执行else下方内容

语句语法格式

if 判断条件:
    执行内容
elif 判断条件:
    执行内容
else:
    执行内容

重复执行语句——while……

示例程序

x = 0

while x != 1:
    x = input("input a number")
    print(x)

print("right")

程序解读

输入1：输出 You are right

输入非1的内容：重新输入

语句语法格式

while 判断条件(一般为相反):
    执行内容

for循环

示例程序

for i in range(10):
    print(i)

程序解读 输出0-9 当使用for循环时，需要使用range函数，range函数的作用是生成一个数字序列，例如range(10)生成一个0-9的数字序列。用于代替while函数进行重复性质的循环

语句语法格式

for 变量 in range(重复执行次数):
    执行内容

提升 类似do……while……

secret_word = "python"
counter = 0

while True:
    word = input("Enter the secret word: ").lower()
    counter = counter + 1
    if word == secret_word:
        break
    if word != secret_word and counter > 7: 
        break

二、数据类型

分类

  • 数字类型：整型 int，单精度浮点型 float，双精度浮点型 double
  • 字符串类型：字符串 str

数据类型转换

案例

x = int(input("Please input a number"))
print(x)

案例解释

  • 输入一个数字，input()函数使用的是字符型，进行比较需将其转换为整型
  • 输出整型数字
  • 字符型整形化int(字符型变量)
  • 复习笔记
