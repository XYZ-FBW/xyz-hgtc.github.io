2024.12.08 C语言语法错误

原版错误代码

#include <stdio.h>
int main(int a, int b)
{
    printf("input a number");
        scanf("%d %d", &a, &b);
        printf("a = %d, b = %d, a + b = %d + %d = %d", a, b,a+b);
        return 0;
}

纠正后代码

#include <stdio.h>
int main(int a, int b)
{
    printf("input a number");
        scanf("%d %d", &a, &b);
        printf("a = %d, b = %d, a + b = %d + %d = %d", a, b,a,b,a+b);
        return 0;
}

错误原因

代码中“%d”和未知数位置没有对应，导致为达到预想效果

错误代码来源：小红书
